Namespace: oapi

oapi

Specification of the base OpenShift API

Set the oshift configuration property to enable these endpoints. If no version is given (i.e. oshift === true) then the version will match the Kubernetes API version.

If enabled, all endpoints are defined as Endpoint instance members of the KubernetesClient object.
