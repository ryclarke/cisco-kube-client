api

Specification of the base Kubernetes API

All endpoints are defined as instance members of the KubernetesClient. The constructor for the client parses each EndpointSpecification defined for the chosen API version to create the appropriate Endpoint object for the client. The specified version parameter for the client must match one of the values defined here.

See:
